常用的linux命令

发表于 2017-02-06

1.ls 查看当前目录下所有的文件和目录。
2.pwd 获得当前工作目录的绝对路径名称。
3.cd 改变当前工作目录。
4.date 显示或者修改系统时间和日期。
5.passwd 用于设置用户的密码。
6.su 用于改变当前用户身份。
7.clear 用来清除屏幕信息。
8.man 显示指定命令的帮助信息
9.who 显示当前登录到系统的用户
10.w 显示当前登录到系统的用户信息

  1. uname 显示操作系统的相关信息
  2. uptime输出系统任务队列信息
  3. last 列出目前与过去登入系统的用户相关信息
  4. dmesg 显示开机信息
  5. free 显示系统内存状态
  6. top 显示系统各个进程的资源占用
  7. ps 显示系统进程在瞬间的运行状态
  8. mkdir 用于创建一个目录 －p指定路径
  9. more 用于显示在屏幕上显示不完的内容
  10. cat 将文件的内容输出到标准输出，还可以合并文件
  11. diff 用来比较文件的差异
  12. grep 根据指定字符串 对文件逐行搜索 若找到这个字符串 酒输出改行内容
  13. rm 用来删除某个目录及其下的所有文件
  14. touch 用来改变指定文件的访问时间和修改时间
  15. ln 用来在文件和目录之间创建链接
  16. file 用来显示文件的类型
  17. cp 用来将给出的文件目录复制到另一个文件或者目录中
  18. find 用来指定的路径下查找指定的文件
  19. split 用来分割文档，将一个文件分成数个
  20. mv 用来移动文件或目录
    压缩与解压
    1.zip/unzip 将一般的文件或者目录进行压缩或者解压，默认生成“.zip” 为后缀的压缩包
    2.gzip/gunzip 将一般的文件进行压缩或者解压。压缩文件的预设的扩展名为“.gz”
    3.tar 对文件或者目录进行打包归档 归档成一个文件 但是并不进行压缩
    4.dd 用来转换或复制文件
    5.cpio 通过重定向的方式将文件进行打包 备份 还原 恢复
    磁盘管理与维护
  21. df 用来检查linux系统的磁盘空间占用情况
  22. du 用来显示文件或目录所占用的磁盘空间情况
  23. fsck 命令用来检查文件系统冰尝试修复错误
  24. 用来强制把内存中的数据写回硬盘，以免数据的丢失
  25. eject 用来退出抽取式设备
  26. mount／umount 挂载以及卸载指定的文件系统
